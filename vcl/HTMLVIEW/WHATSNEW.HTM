What's New in Delphi 1 Version 7.03

Important
Starting with version 7.03, the default compilation no longer will include
the graphics DLL.  Applications which use the the graphics DLL must now
define UseDLL in HTMLCONS.INC
 
Additions to Version 7.03
  • Add support for <NOBR> and <WBR> tags.
Bug fixes, version 7.03
  • GetImageFromFile function onca again handles GIF images.
  • TFileType changed to ThtmlFileType to avoid conflict with Delphi symbol.
  • Visited links now work when using OnStreamRequest.
Bug fixes, version 7.02
  • Missing </TD> and </Table> tags sometimes caused display problems and GPFs.
  • Fix problem where corrupt GIFs caused serious damage.
  • In ThtmlViewer, missing HREF parameter in <A> caused GPFs.
Additions to Version 7.01
  • The <Meta http-equiv="content-type" ...> should now distinguish a number of character sets. I still need information on the codes for some, though.
  • Right click on word gives word without selecting it. See the OnRightClick event in the help file.
  • Changes to accommodate C++Builder 4.
Bug fixes, version 7.01
  • Fix access error in TFrameViewer when a <Meta http-equiv="content-type" ...> statement read.
Additions to Version 7.0
  • New TFrameBrowser component oriented to online use and other protocols.
  • Print Preview for all 32 bit versions.
  • Color change for visited links now supported. See <Body> topic in help file.
  • Color change for links when mouse passes over now supported. See <Body> topic in help file.
  • Image change when mouse passes over (Active Images) now supported.

    Try Me

  • OnDoubleClick event.
  • New htOptions property for ThtmlViewer and extended fvOptions property for TFrameViewer.
Bug fixes, version 7
  • Allow more table columns (now 100)
  • OnImageOver didn't work if image was a link
  • When using drag and drop, sound file didn't work
  • Fix directory problems with meta refresh and <bgsound>
Additions to Version 6.2
  • An OnRightClick event has been added supplying data especially for popup menus.
  • An OnMeta event triggered by <META> tags.
  • Support for the Refresh <META> sequence allows automated document loading.
    • OnMetaRefresh event for ThtmlViewer.
    • fvMetaRefresh option (under fvOptions) for TFrameViewer. Try it.
  • Two additional parameters have been added to the OnFormSubmit event.
Bug fixes, version 6.2
  • The &#nnn; syntax for character references has been expanded to allow leading zeros.
  • The OnSoundRequest event has been added to ThtmlViewer's LoadFromStream, LoadFromBuffer, and LoadStrings methods.
  • Occasionally animated GIFs were displayed as all black.
  • A small vertical alignment error when printing has been fixed.
  • A number of Table and Print fixes.
Additions to Version 6.12
  • New InsertImage method in conjunction with the OnImageRequest Event allows images to be downloaded and inserted when available.
  • More visible cursor.
  • The NoDLL compile time option now supports Jpeg images in the Delphi 3 and 4 versions.
  • NumPrinterPage method indicates the number of pages that will be printed.
  • There is now a Wrap=Off|Soft|Hard attribute for <textarea>.
Bug fixes, version 6.12
  • <SUB> and <SUP> didn't work if first in table cell.
  • Fix the <TD> height attribute which didn't allow for cellpadding.
  • Tabbing now activates Form controls outside of the display area.
  • The VScrollBarRange property now reflects the end of the document more accurately in Delphi 2,3,4 and C++Builder.
Additions to Version 6.1
  • OnSoundRequest event for <BGSound>
  • Transparency supported in palette based PNG images.
  • OnInclude event allows HTML text insertion at load time.
